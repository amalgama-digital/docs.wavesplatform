# [Ride v4 и v3] Int

:warning: Это документация Стандартной библиотеки версии 4 и 3. Рекомендуем использовать версию 5. [Перейти к&nbsp;версии&nbsp;5](/ru/ride/v4/data-types/int)

`Int` — целочисленный [тип данных](/ru/ride/v4/data-types/).

Целочисленная переменная имеет размер 8 байт и хранит целое число из промежутка от -9 223 372 036 854 775 808 до 9 223 372 036 854 755 807 включительно.

## Примеры

``` ride
let age = 42
let length = size("hello")
```
