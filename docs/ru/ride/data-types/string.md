# String

`String` — строковый тип данных.

Строки имеют кодировку [UTF-8](https://ru.wikipedia.org/wiki/UTF-8).

## Ограничения

Максимальный размер переменной типа `String` — 32&nbsp;767 байт (1 символ может занимать до 4 байт).
